백엔드 설계 관련 상세 문서:

- [API_DESIGN.md](./API_DESIGN.md): RESTful API 설계 문서
- [DOMAIN_DESIGN.md](./DOMAIN_DESIGN.md): 도메인 설계 문서
- [DTO_DESIGN.md](./DTO_DESIGN.md): DTO 설계 문서
- [HTTP_STATUS_CODE_DESIGN.md](./HTTP_STATUS_CODE_DESIGN.md): HTTP 상태 코드 및 에러 처리

- [PRESENTATION_GUIDE.md](./PRESENTATION_GUIDE.md): 면접/발표용 설명 가이드

## 빠른 시작

```bash
mvn spring-boot:run
```

서버는 `http://localhost:8080`에서 실행됩니다.
